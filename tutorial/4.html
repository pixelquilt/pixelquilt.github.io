Pixel Quilt Tutorial 4: Loops

Leading by Example

Let's start with some examples that make use of loops and then get into the details

for (x of [0, 1, 2, 3, 4, 5, 6, 7]) { set(x, 0, 255, 0, 0); } for (b of [0, 50, 100, 150, 200, 250]) { set(2, 2, 0, 0, b); wait(1/2); } for (x of [0, 1, 2, 3, 4, 5, 6, 7]) { for (y of [0, 1, 2, 3, 4, 5, 6, 7]) { set(x, y, 0, 128, 255); } } for (x of [0, 1, 2, 3, 4, 5, 6, 7]) { for (y of [0, 1, 2, 3, 4, 5, 6, 7]) { r = x * 15; g = y * 15; b = r + g; set(x, y, r, g, b); } }
